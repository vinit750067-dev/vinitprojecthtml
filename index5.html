★HTML(Boilerplate , filepath Div , Class , ID)

* HTML Boilerplate:

  • HTML boilerplate ek basic template hai jo aapke HTML document ke liye structure provide karta hai.

* Filepath:

  • Filepath ek address hai jo aapke file ko locate karta hai. HTML mein filepath ka use images, links, aur scripts ko include karne ke liye kiya jata hai.

- Absolute filepath: https://example.com/image.jpg

- Relative filepath: ./image.jpg ya ../image.jpg

* Div:

  • div ek container element hai jo content ko group karne ke liye use kiya jata hai.

* Class:

  • Class ek attribute hai jo elements ko group karne ke liye use kiya jata hai. Class ka use CSS aur JavaScript mein elements ko select karne ke liye kiya jata hai.

* ID:

  • ID ek unique attribute hai jo ek element ko identify karta hai. ID ka use CSS aur JavaScript mein elements ko select karne ke liye kiya jata hai.

* Difference between Class and ID:

- Class multiple elements ko group kar sakta hai.

- ID ek unique element ko identify karta hai.
