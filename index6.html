★HTML( Forms)

  • HTML forms ek tarika hai jisse users data input kar sakte hain aur server ko submit kar sakte hain.

* Form Elements:

1. Form Tag: (form) tag form ko define karta hai.
2. Input Tag: (input) tag user input ke liye use hota hai.
3. Label Tag: (label) tag input field ke liye label provide karta hai.
4. Select Tag: (select) tag dropdown list ke liye use hota hai.
5. Textarea Tag: (textarea) tag multi-line input ke liye use hota hai.

* Form Attributes:

1. Action: Form data ko kis URL par submit karna hai.

2. Method: Form data ko kaise submit karna hai (GET ya POST).

* Form Input Types:

1. Text: Single-line text input.
2. Email: Email address input.
3. Password: Password input.
4. Checkbox: Multiple options select karne ke liye.
5. Radio: Single option select karne ke liye.

* Form Validation:

  • Form validation user input ko validate karne ke liye use hoti hai. Yeh client-side ya server-side dono par ho sakti hai.
